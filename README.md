# Simple-interest-calculator
📌 Introduction

This project is a simple interest calculator that calculates the simple interest based on the principal amount, rate of interest, and time period.

📐 Formula

Simple Interest (SI) is calculated using the formula:

SI = (P × R × T) / 100

Where:

P = Principal amount

R = Rate of interest (per year)

T = Time (in years)

🧮 Example Calculation

If:

Principal (P) = 1000

Rate (R) = 5%

Time (T) = 2 years

Then:

SI = (1000 × 5 × 2) / 100
SI = 100
